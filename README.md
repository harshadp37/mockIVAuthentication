# mockIVAuthentication  
  
## Installation Guide :       
1.Open cmd in project folder & **run install**       
2.Start server by **node server.js** OR **nodemon**    

## Description :      
This Project serves the starter code for Authentication System. It includes **SIGN UP**, **SIGN IN** using email id with verification option for email id. Also It provides **FORGOT PASSWORD** AND **CHANGE PASSWORD** after sign in.
